# Design Context

## Product
- Zigns is a signage and template editor for offices, retail, restaurants, schools, facilities teams, and event spaces.
- Users need templates that look production-ready fast, with minimal setup and obvious readability from a distance.

## Brand Tone
- Polished, practical, and trustworthy.
- Clear over clever.
- Modern, but not trendy or gimmicky.
- Friendly enough for general business use, but still disciplined and legible.

## Visual Direction
- Favor photo-led compositions when the message benefits from realism or energy.
- Use strong typographic hierarchy with generous breathing room.
- Keep accent colors restrained and purposeful.
- Prefer editorial, poster-like layouts over generic dashboard blocks.

## Layout Rules
- Never let titles or key copy clip against the canvas edges.
- Use safe margins and intentional alignment, especially for left-aligned list templates.
- Fit long copy to the available space instead of forcing it to overflow or wrap awkwardly.
- Keep one clear focal point per template.

## Typography
- Use bold display faces for headlines and cleaner sans fonts for supporting copy.
- Avoid excessive tracking or oversized type that destroys word shapes.
- If a line is likely to be long, make the text box fit the layout rather than relying on default text sizing.

## Imagery
- Prefer real-world photography over abstract textures when a template needs warmth or context.
- Pexels is an acceptable source for template imagery.
- Match the image subject to the template purpose: meetings for office boards, helmets for safety content, crowds for events, construction sites for notices.

## Quality Bar
- Templates should feel like they were designed intentionally, not auto-arranged.
- Broken spacing, clipped text, or awkward fallback font rendering is a bug, not a stylistic choice.
- If a template is photo-driven, the photo should support the message and not fight the text.
